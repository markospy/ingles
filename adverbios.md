# 📝 Adverbios de Modo (Adverbs of Manner)

> Los adverbios de modo describen **cómo** se realiza la acción del verbo. En español suelen terminar en *-mente* (lentamente, cuidadosamente). En inglés suelen formarse con el adjetivo + **-ly**.

---

## 🎯 Adverbios que modifican el verbo

| 🔤 Palabra    | 🗣️ Pronunciación | 🇪🇸 Significado   | 💡 Ejemplo / Nota        |
| ------------- | ----------------- | ----------------- | ------------------------ |
| Slowly        | slóu-li           | Lentamente        | *Drive slowly.*          |
| Quickly       | kuík-li           | Rápidamente       | *Come quickly!*          |
| Rapidly       | ráe-pid-li        | Rápidamente       | Más formal que *quickly* |
| Carefully     | ké*-ful-li        | Cuidadosamente    | *Cut carefully.*         |
| Gently        | chén-tli          | Suavemente        | *Mix gently.*           |
| Softly        | sóft-li           | Suavemente        | *Speak softly.*         |
| Kindly        | káind-li          | Gentilmente       | *She smiled kindly.*    |
| Loudly        | láud-li           | En voz alta       | *Don't talk loudly.*    |
| Quietly       | kuái-e-tli        | En silencio       | *Close the door quietly.*|
| Well          | uél               | Bien              | *You did well.*         |
| Badly         | báed-li           | Mal               | *He sings badly.*      |
| Easily        | í:-zi-li          | Fácilmente        | *It breaks easily.*     |
| Evenly        | í:-ven-li         | Uniformemente     | *Spread evenly.*        |
| Hard          | já:*d             | Con esfuerzo      | *Work hard.* (sin -ly)  |
| Nervously     | né*-ve*-li        | Nerviosamente     | *She waited nervously.* |

| Still         | estíl             | Todavía / Aún    | *I'm still waiting.*    |

---

## 💡 Uso en la frase

> El adverbio de modo suele ir **después del verbo** o **al final de la oración**.

| Estructura        | Ejemplo                          |
| ----------------- | -------------------------------- |
| Sujeto + verbo + adverbio | *She walked **slowly**.*        |
| Verbo + objeto + adverbio | *He closed the door **quietly**.* |
| Imperativo + adverbio    | ***Stir** the rice **gently**.*  |

---

## 🔗 Relación adjetivo → adverbio (-ly)

| Adjetivo · Pronunciación | Adverbio · Pronunciación |
| ------------------------ | ------------------------- |
| Slow · `slóu`            | Slowly · `slóu-li`        |
| Careful · `ké*-ful`      | Carefully · `ké*-ful-li`  |
| Quick · `kuík`           | Quickly · `kuík-li`        |
| Gentle · `chén-tel`      | Gently · `chén-tli`        |
| Kind · `káind`           | Kindly · `káind-li`        |
