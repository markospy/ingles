# Preposiciones

| -------------- | ----------------- | ----------------------------------------- | ------------------------- |

| 📍 On          | ón                | Sobre / Encima de (tocando)               |                           |
| 📦 Under       | án-de*            | Debajo de                                 |                           |
| ⬅️ In front of | in fránt ov       | Delante de                                |                           |
| ➡️ Behind      | bi-jáind          | Detrás de                                 |                           |
| 🫂 Next to     | nékst tu          | Al lado de                                |                           |
| ↔️ Between     | bi-tuí:n          | Entre (dos cosas)                         |                           |
| ☁️ Above       | a-báv             | Por encima de (sin tocar)                 |                           |
| 📉 Below       | bi-lóu            | Por debajo de                             |                           |
| 🆙 Over        | óu-ve*            | Sobre / Por encima                        |                           |
| 🏘️ Among      | a-máng            | Entre (varios/un grupo)                   |                           |
| 🏠 At          | áet               | En (un punto específico/lugar)            |                           |
| ↘️ Near        | ní:*              | Cerca de                                  |                           |
| 🏃 Far from    | fá:* fram         | Lejos de                                  |                           |
